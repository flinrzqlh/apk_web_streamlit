Web Application Course - Streamlit Projects & Learning Files

Welcome to my repository! This is where I store all my working and learning files for my Web Application Course, primarily focusing on Streamlit and Machine Learning concepts. Here you’ll find various projects and documentation that reflect my journey through this course.

This repository contains the following:
#awworks - This folder holds the core of my work. All my Streamlit projects and related working files can be found here.
aplikasi_web - This folder is dedicated to documentation. It includes PDFs, Word documents, and reports related to the course.
.gitignore - A .gitignore file is used to ignore the virtual environment folder (testenv) in my local setup, ensuring that irrelevant files don't clutter the repository.

About the Course:
The main focus of this course is using Streamlit to develop web applications, with a particular emphasis on incorporating machine learning into those applications. Throughout this repository, you’ll find examples and exercises that demonstrate various features and use cases of Streamlit, as well as how machine learning models can be deployed within a web app.

Key Technologies:
Streamlit: An easy-to-use Python framework for creating web applications.
Machine Learning: Integrated within some of the projects to perform predictive tasks.
How to Use This Repository
Feel free to explore the projects and documentation within the repository. Each project within #awworks comes with its own brief description in the code or a separate markdown file to guide you through the process.
